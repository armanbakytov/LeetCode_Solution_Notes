There are `n` soldiers standing in a line. Each soldier is assigned a **unique**`rating` value.

You have to form a team of 3 soldiers amongst them under the following rules:

- Choose 3 soldiers with index (`i`, `j`, `k`) with rating (`rating[i]`, `rating[j]`, `rating[k]`).
- A team is valid if: (`rating[i] < rating[j] < rating[k]`) or (`rating[i] > rating[j] > rating[k]`) where (`0 <= i < j < k < n`).

Return the number of teams you can form given the conditions. (soldiers can be part of multiple teams).

**Example 1:**

**Input:** rating = [2,5,3,4,1]
**Output:** 3
**Explanation:** We can form three teams given the conditions. (2,3,4), (5,4,1), (5,3,1). 

**Example 2:**

**Input:** rating = [2,1,3]
**Output:** 0
**Explanation:** We can't form any team given the conditions.

**Example 3:**

**Input:** rating = [1,2,3,4]
**Output:** 4

**Constraints:**

- `n == rating.length`
- `3 <= n <= 1000`
- `1 <= rating[i] <= 105`
- All the integers in `rating` are **unique**.

# Solution
>I haven't written the though process. But we just take the middle element j and see the possible combinations on the left and right. And then multiply.
```java
class Solution {  
    public int numTeams(int[] rating) {  
        int sum = 0;  
        for(int i = 0; i < rating.length; i++){  
            int[] left = lookLeft(i, rating);  
            int[] right = lookRight(i, rating);  
            sum += left[0]*right[1] + left[1] * right[0];  
        }  
        return sum;  
    }  
  
    public int[] lookRight(int n, int[] list){  
        int bigger = 0;  
        int smaller = 0;  
        for(int i = n+1; i< list.length; i++){  
            if(list[n] < list[i]) bigger++;  
            else smaller++;  
        }  
        return new int[]{smaller, bigger};  
    }  
  
    public int[] lookLeft(int n, int[] list){  
        int bigger = 0;  
        int smaller = 0;  
        for(int i = n-1; i >= 0; i--){  
            if(list[n] < list[i]) bigger++;  
            else smaller++;  
        }  
        return new int[]{smaller, bigger};  
    }  
}
```
